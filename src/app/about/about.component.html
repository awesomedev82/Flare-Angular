Project Flare

Project Flare is a full-fledged Tinder web client made in Angular with extra features such as the ability to see profile picture success rate (If Smart Photos is enabled), See possible matches, Search profile on Facebook/Instagram, an input for images to detect specific face attributes and emotions such as age, gender, happiness, etc...




This project was made with Angular and NodeJS


FAQ

Profiles

What is this "Possible matches" feature?

Clicking this will set your distance to 1 mile, anyone who shows up with a distance further than 1 mile will be a possible match, this will be shown with a green "Possible match" text on the profile itself. If it doesn't show anything you don't have any new matches


Passport

Why can I not use the Passport function?

You need to have an active subscription for Tinder Plus in order to use Passport to change your travel location
